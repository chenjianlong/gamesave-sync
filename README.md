# gamesave syncing

Syncing game save between PC via S3

[简体中文文档](README-zh_CN.md)

## Supported games

* The Witcher 3
* Skyrim
* NewPAL
* Wind Fantasy 3
* Wind Fantasy 4
* Wind Fantasy 5
* Wind Fantasy 6
* Wind Fantasy XX

## Supported OS

* Windows 10

## How to use

* Fill config.ini
* Use go build generate gamesavesyncing.exe
* Run gamesavesyncing.exe
